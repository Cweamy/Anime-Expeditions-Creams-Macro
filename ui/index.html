Display Scale Warning ×
Your Windows display scale is -, not 100%. Every click/search position this macro uses was captured at 100% scale, so anything else is a common cause of clicks landing slightly wrong or buttons not being found.
Fix: Windows Settings > System > Display > Scale — set it to 100%, then restart your computer (not just the macro) — Windows and Roblox don't always fully pick up a scale change until everything's been relaunched fresh.
If clicks/detection are still off after that, also try: set the game's own UI scale to 1x (in-game Settings), turn off HDR (Windows Settings > Display), and if your monitor runs a high refresh rate, try dropping it under 100Hz — some setups render color slightly differently at very high refresh rates, which can throw off the macro's image matching.
